# Sense-Making Vault

Personal knowledge graph for capture → label → recall.

This is an **Obsidian-compatible** plain-markdown vault. Open it in Obsidian by pointing the app at this directory. The capture pipeline (run by Zo) writes here; the user does not have to file things by hand.

## Folders

| Folder | What goes here |
|---|---|
| `inbox/` | Captures Zo wasn't sure how to classify. Triage by hand or with Zo. |
| `notes/` | Free-text thoughts, ideas, working notes. |
| `links/` | Captured URLs (one note per link, with summary). |
| `quotes/` | Snippets from books, articles, conversations. |
| `people/` | One note per person. Filename is the person's display name. |
| `media/` | Photos and other binary files + a `.md` sidecar with metadata. |
| `journal/` | `YYYY-MM-DD.md` daily log auto-appended on every capture. |

## Conventions

- All notes use the frontmatter schema in `AGENTS.md`.
- Links between notes use Obsidian `[[wikilinks]]`.
- Filenames: `YYYY-MM-DD-slug.md` for dated items; `Person Name.md` for people.

See `AGENTS.md` for the full spec.
